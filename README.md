# Mind Journal App

## Description
Mind Journal App is a Flutter-based mobile application that helps users practice mindfulness and reflection through guided journaling sessions.

Users can start a session, answer reflection questions, and review their thoughts.

## Features
- Start journaling sessions
- Reflection questions
- Simple and clean user interface
- Smooth navigation between screens
- Built using Flutter

## Technologies Used
- Flutter
- Dart
- Material Design

## How to Run the App

1. Clone the repository

git clone https://github.com/ashitashinde96-gif/MindJournalApp_New

2. Open the project folder

3. Run the command

flutter run

## APK
The release APK file is included for installing the app directly on Android devices.
